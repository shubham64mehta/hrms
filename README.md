# HRMS Lite Frontend

## Project Overview

This is the frontend for the **HRMS Lite application**, a lightweight Human Resource Management System.  
It provides a clean and user-friendly interface for:

- Managing employee records
- Marking and viewing attendance

The frontend is built using React with a focus on simplicity, usability, and a professional UI.

---

## Tech Stack Used

- React (Create React App)
- Tailwind CSS
- Axios
- TanStack React Query

---

## Steps to Run Locally

### 1. Clone the repository

```bash
git clone <your-repo-url>
cd frontend
```

### 2. install dependencies

npm install

### 3. run the application

npm start

### 4. Access the app

http://localhost:3000
